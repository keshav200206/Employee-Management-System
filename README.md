# Employee-Management-System
This project aims to develop a model for maintaining a list of employee's in the company. It helps to manage the details of the employees in the database and improve productivity.

## Code Description 
The Source code written in Java using Java Swing and Java AWT for building interactive Graphical User Interfaces (GUI).

## Installation
1. Ensure you have Java Package installed on your machine.
2. Ensure you have MySQL Package installed on your machine.
3. Install the required libraries

    3.1 MySQL-Connector-Java jar
   
    3.2 jcalendar jar.
   
    3.3 rs2xml jar.

4. Execute the code in Java Environment with MySQL.
